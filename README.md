# IBuySpy-Dev Organization Site

This repo powers the IBuySpy-Dev organization landing page at https://ibuyspy-dev.github.io/

## What's on the page

- **Hero** — IBuySpy-Dev heading, tagline, links to Migration Factory and GitHub org
- **What We Build** — Migration Factory, Research & Tooling, Reference Apps
- **Featured Projects** — Cards for all key repos (app-migration-with-ai, spoke repos, ibuypets-v3)
- **Migration Factory callout** — static summary linking to the live factory dashboard
- **Footer** — links to org and dashboard

## Factory Dashboard

The live factory shop floor board (all 55 apps, station details) is at:
→ https://ibuyspy-dev.github.io/app-migration-with-ai/

## Local Development

Open `index.html` directly in a browser — no server or build step required.

## Deployment

Automatically deployed to GitHub Pages on push to `main`.
